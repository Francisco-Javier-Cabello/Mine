NAVIGATION

Structure and components

Structure Material de aprendizaje

  1. Doctype declaration

    2. Or the doctype declaration

    This is the very first line has to use in every single HTML document and it tells the web browser which type of document it is
    and defines the version of HTML that you will be using.

    Allows for the use of markup validators

  2. The Head

    3. The head is the first portion of the html structured document

    Contains information and metadata that is not part of the content of the page

    Enhances the content

    • Metadata tags
    • Links to external style sheets
    • Internal styling
    • Title of the web page to appear in the browser window

  3. The body

    4. Is the portion that defines all the important content of your web page

    What you want your visitors to see is all found in the body

    Includes structure to define the content (Various elements and attributes

    • Headings
    • Paragraphs
    • Sections
    • Lists
    • Images and media

    Can include styling of the content

  4. The footer

    5. It's not a required part of a HTML document

    Appears at the bottom of the HTML web page

    It's visible for the visitors

    Typically defines information about the section

    • Author
    • Related content
    • Copyright data
    • Navigation elements

    A webpage can have multiple footers if necessary

Components

  1. Tags
    • p
    • h1
    • em
  2. Elements
    • Block (creates separated sections on the page)
    • Inline (acts within a section of the content)
  3. Attributes
    • for example style="font-family: Verdana:"

BASIC RULES FOR STYLING

Give same format to all the tags of the same type

Example:

  • Inside the head element insert a style attribute
  • Define which tag you want to give the style rules
  • Set the rules (font, size, color, etc...)
  • Don't forget to close the brackets and the style attribute
